# MazeRace

A 2D competitive maze racing game built with Godot 4 and GDScript.

## Current Features

- Two-player local gameplay
- Independent WASD and Arrow Key controls
- Reusable player scene
- Git version control

## Planned Features

- Handcrafted maze
- Procedural maze generation
- Goal detection
- Win/Draw system
- Sound effects
- Main menu
- Multiple maze sizes

## Built With

- Godot 4.6.1
- GDScript
